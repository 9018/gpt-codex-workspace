## CodexPro Apps SDK card pattern

Repository: https://github.com/rebel0789/codexpro

License: MIT License

Copyright (c) 2026 CodexPro contributors

GPTWork's Apps SDK card subsystem adapts the CodexPro card infrastructure pattern: a single versioned `ui://` widget resource URI, `text/html;profile=mcp-app` resource MIME type, centralized descriptor metadata using `_meta.ui.resourceUri` and `_meta["openai/outputTemplate"]`, resource metadata that includes both standard MCP Apps fields and ChatGPT compatibility fields, and bounded structured tool results for compact widget rendering.

No CodexPro branding, logos, CLI, workspace, tunnel, or unrelated product functionality was copied. The implementation remains GPTWork-branded and scoped to Apps SDK card registration, resource metadata, result tagging, and widget hydration behavior.
